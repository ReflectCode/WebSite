ReflectCode

Automated source code transformation service leveraging AI + ML technologies

ReflectCode leverages Artificial Intelligence (AI) and Machine Learning (ML) technologies to generate value addition from existing source code.

Currently we are focusing on Digital Assets and UI Layout transformation from existing Android project.

Our long term goal is to transform Android studio project into semantically equivalent Xcode project for iOS, which can be directly compiled to iOS App. 

Overview about service

Overview about porting for various types of files in a Android project

1

Digital Assets Transformation

As a part of Digital Assets Transformation offering, RC supports conversion of following files -
1. Assets (jpg, .png, .9.png) to "Assets.xcassets" 
2. Mipmap images to “.appiconset”
3. Color xml to “.colorset”
4. Shape xml to PDF format in asset catalog
5. Vector xml files to Swift Code (CAShapeLayer) 
6. Animation xml file

Check our demo project on GitHub

2

We are working to add support for

1. Port UI Layout xml files to storyboard file
2. Resource xml files 
3. Test case generation 
4. Vector Animation xml file

3

Our plans to add support in near future

1. Port Java POJO Classes to Swift
2. Port Java code into swift code  
3. Support Java Libs e.g. GSON, Glide, Retrofit2 etc.
