ReflectCode

Automated source code transformation service leveraging AI + ML technologies

RC leverages Artificial Intelligence (AI) and Machine Learning (ML) technologies to generate value addition from existing source code e.g. Digital Assets transformation, Test Cases creation, Statistical information, and complete source code porting.

Currently we are focusing on digital assets conversion and test case generation from existing Android project.

Our long term goal is to support conversion of Android studio project into semantically equivalent Xcode project for iOS which can be directly compiled into iOS App. 

Benefits of using RC

As compared to other developemnt tools, RC comes with unique benefits

No new technology

You work with standard techlology and tools 
Android (Android Studio, Java) 
iOS (xCode, Swift, Storyboard)

No dependency

Assurance that your product is not affected, if third party tool is out of business or if their license, terms / conditions are changed against you

No Limitations

You are free to add any new control / library to converted project. No lowest common denominator restriction  

Save efforts

As iOS project is generated automatically, you save efforts of developing iOS code from scratch
